# React 19 UI Pro Skill

Documentation of premium aesthetic standards, components, and layout contracts. This file acts as runtime documentation for AI agents.

## Core Directives
1. Use React 19 rules and features like optimized `useActionState` where applicable.
2. Adhere to deep navy (`#0B2545`), mid blue (`#1B4F8A`), ice/tech blue (`#00AEEF`), Light ice (`#E8F4FD`) color guidelines.
3. Keep the typography sophisticated:
   - Display headings: "Syne" font (engineered/industrial look).
   - Body typography: "DM Sans" (highly readable).
   - Tech specs / stats: "JetBrains Mono".
4. Layout must include diagonal dividers, frosted-glass statistics, and smooth infinite marquees.
